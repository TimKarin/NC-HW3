File:

File must be a collection of lines.
Each line contains information about one user.
Information is separated by sign #.
Information is:

  • first Name: 3-20 symbols
  • last Name: 3-20 symbols
  • Birthday: format YYYY-MM-DD
  • Salary: >0
  • E-mail
  • Company Name: 3-40 symbols

File example:
Ivanov#Ivan#2000-01-01#50000#ii@mail.ru#SomeCompany
Petrov#Petr#1980-12-21#20000#pp@gmail.com#Company
